## 📐 Project Scope: Architecture & Design
This project is a **Cloud Design Showcase**. It focuses on the theoretical and architectural planning phase of a cloud migration:
* **Architecture Diagramming:** Visualizing the interaction between AWS services.
* **Resource Planning:** Choosing the right instance types and database engines (EC2 & RDS).
* **Networking & Security Design:** Planning the VPC, Subnets, and Security Group rules for data isolation.
* **Cost Analysis:** Estimating the monthly operational costs using AWS Pricing Calculator.

*Note: This repository represents the design and planning phase; no live infrastructure was deployed on the AWS Console for this specific project.*
